<p align="center">

## Getting Started

### Clone
* Clone esse projeto em sua máquina através do comando:
  ```
  $ git clone https://github.com/LeonMarqs/nlw-ecoleta.git
  ```

### Built With
* NodeJS
* ReactJS
* React-Native
* Sqlite

### Running

* Server
  ```
  $ npm run dev
  ```

* Web
  ```
  $ npm start
  ```

* Mobile
  ```
  $ npm start
  ```

### Other Dependences
* As outras dependências estão no arquivo `package.json` de cada pasta.
